# AI Resume Analyzer using n8n

AI-powered resume analysis workflow built using n8n, OpenAI, and Google Sheets.

## Features

- Resume strengths analysis
- Weakness detection
- Career recommendations
- AI-generated resume feedback
- Google Sheets automation

## Tech Stack

- n8n
- OpenAI API
- Google Sheets

## Workflow

1. User enters resume data
2. OpenAI analyzes the resume
3. AI generates improvements and career suggestions
4. Results are automatically stored in Google Sheets
